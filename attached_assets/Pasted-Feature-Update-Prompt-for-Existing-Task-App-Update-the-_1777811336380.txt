2. Recurring Logic

When a recurring task is marked Done, the system should automatically create the next cycle.

The new task should:

move back to To Do
keep the same task title and settings
reset status
update deadline

Rules:

Daily Task

If recurrence_type = daily

New cycle starts every day at 00:00

Example:
Complete task today → new task appears tomorrow at 00:00.

Weekly Task

If recurrence_type = weekly

User selects a weekday (example Sunday).

New cycle starts at 00:00 on that day.

Example:

Weekly review
Completed Sunday → next task appears next Sunday 00:00.

Monthly Task

If recurrence_type = monthly

User selects a day of month.

Example:

Pay bills
If set to 1st → new task appears every 1st at 00:00.

Custom Recurrence

Allow user to define:

Repeat every X days

Example:

Every 3 days
Every 10 days
