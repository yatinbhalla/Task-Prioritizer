Build a modern mobile task prioritization app that helps users decide what to work on next by automatically adjusting task priority based on deadline proximity and urgency.
The app should feel like a simplified mix of Trello and Todoist, optimized for personal productivity.
________________________________________
Platform
Create a cross platform mobile app (Android) using React Native with Expo.
Use clean modular architecture and maintainable code.
Store data locally using local storage.
________________________________________
Core Idea
The app helps users manage tasks by combining:
•	Manual priority
•	Deadline proximity
•	Automatic urgency scoring
Tasks should become more important automatically as deadlines approach.
The app should visually guide the user to what must be done next.

Medium

To Do
In Progress

________________________________________
Smart Priority System
The app should calculate a dynamic priority using deadline proximity.
Example rules:
If deadline > 7 days
→ use manual priority
If deadline ≤ 5 days
→ increase priority by one level
If deadline ≤ 2 days
→ increase priority by two levels
If deadline ≤ 24 hours
→ automatically set priority to Top
If overdue
→ mark as Critical
Priority escalation order:
Low → Medium → High → Top → Critical
This calculation should refresh every time the app opens and once per day.
________________________________________
Urgency Score
Calculate a numeric urgency score so tasks can be sorted intelligently.
Example formula:
urgency_score =
(priority_weight × manual_priority)
+
(deadline_weight × days_remaining_inverse)
Where:
Low = 1
Medium = 2
High = 3
Top = 4
Critical = 5
Tasks with higher urgency score appear first.
